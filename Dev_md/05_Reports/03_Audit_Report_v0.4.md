## 3. 주요 결함 및 조치 내역 (Defects & Resolutions)

### 3.1 [UI] 카드 이미지 잘림 및 텍스트 이탈
*   **증상:** `background-size: cover` 사용 시 카드 테두리와 텍스트가 잘리는 현상.
*   **조치:** CSS `background-size: 100% 100%`로 변경하여 비율 강제 맞춤으로 해결.

### 3.2 [UX] 부채꼴 내 카드 역방향 표시
*   **증상:** 부채꼴 배치 시 카드가 180도 뒤집혀서 출력됨.
*   **조치:** `transform-origin` 좌표 수정 및 `rotate()` 각도 계산식 보정으로 정방향 아치 구현.

### 3.3 [Layout] 슬롯 수직 나열
*   **증상:** 뽑힌 카드가 가로가 아닌 세로로 길게 나열됨.
*   **조치:** `.slots-container`에 `flex-direction: row` 및 `flex-wrap: wrap` 속성 명시.

## 4. 권고 사항 (Recommendations)
1.  **배포 환경 표준화:** 현재 로컬(`localhost`) 환경에서만 검증되었으므로, **Docker 컨테이너**를 통한 배포 패키징이 시급함.
2.  **자동화 테스트:** UI 기능은 수동 테스트로 검증되었으나, 지속적인 품질 관리를 위해 **E2E(End-to-End) 테스트** 스크립트 도입 권장.
3.  **DB 영속성 점검:** 세션 데이터가 DB에 정상적으로 저장되고 있는지 확인 필요 (현재는 In-memory 캐시 의존도가 높을 수 있음).
